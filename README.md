# Python Project Portfolio

## Overview
This repository contains a collection of Python projects demonstrating various programming concepts and practical applications. Each project is organized in its own directory and showcases different Python skills and libraries.

## Project List

### 1. Beginner Python Projects
- **Guess the Number Game** - A simple number guessing game with difficulty levels
- **Rock Paper Scissors** - Classic game implementation with score tracking
- **Password Generator** - Creates secure passwords with customizable parameters
- **Hangman** - Word guessing game with visual feedback

### 2. Intermediate Python Projects
- **Snake Game** - Classic Snake game with GUI using Turtle graphics
- **Pong Game** - Two-player Pong implementation
- **Turtle Crossing** - Road crossing game with increasing difficulty
- **Quiz App** - Text-based quiz with question database
- **U.S. States Game** - Educational geography game

### 3. Advanced Python Projects
- **Password Manager** - Desktop app to store and generate passwords
- **Pomodoro Timer** - Productivity timer with work/break intervals
- **Flash Card App** - Learning tool with progress tracking
- **Automated Email Sender** - Sends emails on specified dates
- **Weather App** - Fetches and displays weather data using API

### 4. Data Science Projects
- **Stock Price Tracker** - Visualizes stock data with matplotlib
- **Movie Recommendation System** - Basic recommendation algorithm
- **Data Analysis Tool** - Performs analysis on CSV datasets

## Technologies Used
- **Python 3.x**
- **Tkinter** - GUI development
- **Turtle** - Graphics for game development
- **Pandas** - Data manipulation
- **Requests** - API integration
